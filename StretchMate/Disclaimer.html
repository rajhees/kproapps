Description of App:

The App provides detailed description of basic through to advanced exercises including:

  • Stretching exercises for specific muscle groups
  • Strengthening exercises for specific muscle groups
  • Exercises for mobilising joints (increasing mobility)
  • Core strengthening exercises
  • Pilates based exercises
  • Exercises using no or minimal equipment (swiss ball, foam roller, resistance band, spiky ball)
  • Popular programs consisting of exercises relevant for the specific program

The user can save favourite exercises and devise their own program or their health care professional can devise a program and control this from their own software. There is an alarm setting whereby the user or health professional can send reminders for exercise completion. Exercises can be recorded when completed. The aim of the App is:

1) to improve and increase patient compliance with recommended exercises from their health care professional; and

2) Help the user to do exercises which would be specifically beneficial to their lifestyle or specific injury by allowing the user to have the exercises and description with them at all times (on their mobile phone).
